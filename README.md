# Kirs_Mikhail_OOP_Assignment

## The Beatles
![The Beatles](images/the-beatles.avif)

## Overview  
This project demonstrates Object Oriented Programming concepts in PHP by building a hierarchy of musician classes inspired by The Beatles.  

At the beginning of this assignment I was observing the Week 4 class example and doing my own research to better understand how inheritance, traits and interfaces work together. The next step for me was to build my own hierarchy based on The Beatles while also adding the necessary improvements required by the assignment instructions.  

I decided to dedicate this assignment to the Liverpool band because in the last couple of weeks they served as a great inspiration for me and helped me to feel a little bit better.

The goal of this project is to show understanding of OOP structure, hierarchy depth and reusable functionality.   

---

## Features  
- Multi-level class hierarchy  
- Method overriding for different musician roles  
- Shared functionality using a trait  
- Interface implementation  
- Autoloading with `spl_autoload_register`  
- Simple testing of objects in `index.php`  

---

## Tech Stack  
- **Language:** PHP  
- **Concepts Used:** Inheritance, Traits, Interfaces, Polymorphism  
- **Version Control:** Git & GitHub  

---

## Structure  
The project consists of multiple class levels:

- **Parent:** Musician  
- **Second Level:** Vocalist, Guitarist, Drummer  
- **Third Level:** PaulMcCartney, JohnLennon, GeorgeHarrison, RingoStarr  

Additional components include:  
- SongwriterTrait  
- PerformerInterface
